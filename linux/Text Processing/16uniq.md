# 🔁 Příkaz uniq

```uniq``` je velmi užitečný command-line program pro zpracování textu.  

Pomáhá při kontrole a manipulaci s textovými soubory porovnáváním nebo filtrováním opakujících se řádků, které jsou vedle sebe.  

Ať už pracujete se seznamem dat nebo s velkým textovým dokumentem, příkaz ```uniq``` vám umožní najít a odfiltrovat duplicitní řádky nebo dokonce poskytnout počet jednotlivých jedinečných řádků v souboru.  

Je důležité si uvědomit, že ```uniq``` odstraňuje pouze duplicity, které jsou vedle sebe, takže aby bylo možné tento příkaz využít na maximum, data se často nejprve třídí pomocí příkazu ```sort```.
