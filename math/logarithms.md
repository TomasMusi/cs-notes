# Logaritmy

## 1. Základy mocnin (exponentů)

Musíš chápat:
- Co je mocnina

aⁿ = a · a · ... · a (n-krát)

3⁴  
- 3 = základ  
- 4 = exponent (mocnitel) říká, kolikrát násobíme  

## 2. Pravidla mocnin

### 1. Když násobíš dvě stejné základy:

aᵐ · aⁿ = aᵐ⁺ⁿ

Když máš 2³ · 2⁴ → máš (2·2·2) · (2·2·2·2)  

To je 7 dvojek = 2⁷

2³ · 2⁴ = 2³⁺⁴ = 2⁷

**Sečteš exponenty**

### 2. Když dělíš dvě stejné základy:

aᵐ / aⁿ = aᵐ⁻ⁿ

Například:

2⁵ / 2² = (2·2·2·2·2) / (2·2) = 2³

2⁵ / 2² = 2⁵⁻² = 2³

**Odečteš exponenty**

### 3. Když mocníš mocninu:

(aᵐ)ⁿ = aᵐ·ⁿ

Například:

(2³)² = (2³)·(2³) = 2³⁺³ = 2⁶  
nebo (2³)² = 2³·² = 2⁶

**Vynásobíš exponenty**

### 4. Každé číslo na nultou = 1:

a⁰ = 1

### 5. Záporný exponent znamená „obráceně“:

a⁻ⁿ = 1 / aⁿ

Například:

2⁻³ = 1 / 2³ = 1 / 8
