# Měření úhlů

Úhel se dá měřit **ve stupních (°)** nebo **v radiánech (rad)**.

- **Kladný úhel** se měří **proti směru hodinových ručiček**.  
- **Záporný úhel** se měří **po směru hodinových ručiček**.

## Převod mezi stupni a radiány

$$ 180° = \pi \text{ rad} $$

Z toho vyplývá:

$$1° = \frac{\pi}{180} \quad \text{a} \quad 1 \text{ rad} = \frac{180}{\pi}°$$

## Tabulka převodu

| Stupně | Radiány |
| ------ | -------- |
| 0°     | 0 |
| 30°    | $$\frac{\pi}{6}$$ |
| 45°    | $$\frac{\pi}{4}$$ |
| 60°    | $$\frac{\pi}{3}$$ |
| 90°    | $$\frac{\pi}{2}$$ |
| 180°   | $$\pi$$ |
| 360°   | $$2\pi$$ |
