# Español – Preposiciones y adverbios

## 1. Preposiciones (předložky)
- **en** = v, na  
- **de** = z, od  
- **a** = do, k  
- **con** = s  
- **sin** = bez  
- **por** = kvůli, přes, skrz  
- **para** = pro, aby  
- **sobre** = na, o (něčem)  
- **entre** = mezi  

---

## 2. Adverbios de lugar (příslovce místa)
- **aquí** = tady  
- **allí** = tam  

---

## 3. Adverbios de tiempo (příslovce času)
- **siempre** = vždy  
- **nunca** = nikdy  
- **ahora** = teď  
- **hoy** = dnes  
- **mañana** = zítra
